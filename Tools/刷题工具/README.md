---
Name: 通用题库自动生成刷题工具
Description: 基于`Python`开发用于给定题库转化为刷题模式进行记忆，例如习概等需要大量刷题记忆的科目，可导入学校或出题方给定题库，格式包括`docx`、`markdown`、`json`、`txt`文档，值得注意的是`pdf`文档直接导入可能出现识别问题，建议转换为`docx`文档后导入。
---

## 使用方式
1. **本地浏览器直接打开**:`通用刷题工具.html`
2. **执行Python程序打开**:`http://127.0.0.1:8765/通用刷题工具.html`

## 工作流程
1. `docx`导入
   - 读取 .docx，本质上它是 zip 包，从里面取 word/document.xml，提取段落文字。
2. `txt/md`导入
   - 直接读取纯文本
3. `pdf`导入
   - 读取 .pdf，通过脚本工具从里面取 word/document.xml，提取段落文字再输出为`txt/md`文件。
4. 文本识别题目
   - 按 `1.`、`1．`、`1、`、`1)` 这类题号切块。
5. 识别章节
   - 识别 `# 第一章`、`《第一章...》`、`第一章...`、`导论`、`绪论` 等标题。
6. 识别题型
   - 如果附近出现`“单选题/多选题/判断题”`，就作为题型提示。
   - 如果没有提示，就根据答案判断：答案多于 1 个字母就是多选，`对/错/正确/错误/√/×` 就是判断。
7. 识别选项
   - 识别 `A.`、`A．`、`A、`、`A)` 、`A 空格`这类选项。
8. 识别答案
   - 题干内：例如`中国特色社会主义最本质的特征(C)`。
   - 题目末尾：`答案：C`、`答案：AB`、`答案：错`。

### V1.2 版本(新增导入预览功能)
导入流程修改:
1. 点 `导入题库`
2. 选择文件或粘贴文本
3. 点 `解析导入`
4. 进入 `解析预览`
5. 在预览列表里逐题`编辑`或`删除`
6. 点 `确认导入并开始刷题`
7. 切换当前题库

### V1.3 版本(新增题库管理功能)
新增功能：
- 新增 `保存题库`
- 新增 `题库管理`
- 可把当前题库保存到浏览器本地
- 题库管理面板可查看本地题库列表
- 可加载已保存题库
- 可直接改名并点 保存名称
- 可删除本地保存的题库
- 已保存题库加载后，做题进度按题库 id 区分，重命名不丢进度
